## 关键发现

### 1. 前期性能超越实验9

| 指标 | 实验9 (SEWN) | 实验10 (消融B) | 对比 |
|------|-------------|----------------|------|
| 最佳Val Loss | 0.0006 | **0.0001** | **6倍提升** |
| 达到最佳Epoch | 8 | **3** | **2.7倍加速** |

**结论**: 无元认知层在简单任务上收敛更快、性能更好。

### 2. 训练崩溃现象

**崩溃时间**: Epoch 5 开始时  
**崩溃原因**: 数值不稳定导致梯度爆炸

#### 崩溃机制分析

```
Epoch 4: Loss = 0.0001 (极低)
         ↓
梯度 ∂Loss/∂params 变得极端
         ↓
固定学习率 8e-5 过大
         ↓
权重更新幅度超出数值稳定范围
         ↓
Epoch 5: Loss = NaN (崩溃)
```

#### 实验9为何未崩溃？

```python
# 实验9 的隐式保护
meta_loss = F.mse_loss(confidence_tensor, correct_mask.mean())
loss = loss + 0.1 * meta_loss  # 正则化效果
```

元认知层产生的 `meta_loss` 提供了**隐式正则化**：
- 防止 Loss 过度下降
- 提供额外的约束信号
- 动态调节训练过程

### 3. 元认知层的真正价值

| 传统认知 | 本实验发现 |
|----------|-----------|
| 元认知层提升性能 | ❌ 前期反而降低性能 |
| 元认知层加速收敛 | ❌ 收敛更慢 |
| **元认知层是训练稳定器** | ✅ **防止崩溃** |

**核心结论**: 元认知层的价值不在于性能提升，而在于**训练稳定性保障**。
